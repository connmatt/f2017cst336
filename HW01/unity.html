Unity is a cross-platform game engine, which is primarily used to develop video games for computers, consoles, and mobile devices.

Unity was first announced back in 2005 at Apple's Worldwide Developers Conference so developers could make games specifically for OS X.

Unity has extended their target to 27 platforms.

C# and Javascript are the two languages they use in their engine.



Games that have been made using Unity:

  • Rick and Morty: Virtual Rick-ality
  • Cuphead
  • Hearthstone: Heroes of Warcraft
  • Digimon World: Next Order
  • Crossy Road
  • Temple Run
